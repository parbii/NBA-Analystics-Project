# NBA Data Analytics & Predictive Modeling

## Overview
This project focuses on leveraging data-driven analytics to identify high-probability outcomes for daily NBA investments. By synthesizing real-time player statistics, team trends, and historical performance data, this model aims to remove emotional bias from the decision-making process.

## Analytical Methodology: The 12-Filter System
The core of this project is a proprietary **12-filter logic** used to evaluate player props and game totals. 
* **Volume Analysis:** Evaluating minutes played and usage rates.
* **Consistency Metrics:** Tracking performance over the Last 5, Last 10, and Season-to-Date.
* **Matchup Optimization:** Cross-referencing defensive rankings vs. specific positions.
* **Market Discrepancy:** Identifying "edge" by comparing modeled outcomes against platform data (Outlier).

## Tech & Data Stack
* **Primary Tools:** Statmuse (Deep-querying historical data), Outlier (Market trend analysis).
* **Data Logic:** Developed using logical operators and statistical filtering.
* **Documentation:** Managed via VS Code.

## Key Insights
* Developed a structured workflow for daily "investment" reports.
* Utilized advanced querying to isolate variables like "Rest vs. No Rest" and "Home/Away Splits."
* Focused on sustainable ROI through data-backed risk management.
